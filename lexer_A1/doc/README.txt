Pre installation
1. Install ply through `pip3 install ply`
2. If you are unable to install, Download from `http://www.dabeaz.com/ply/index.html` into src directory

Run in main directory as :
	`python3 ./src/lex.py --cfg=./tests/cfg1/cfg1.txt ./tests/input1/input2.cpp --output=./tests/output/output1.html`
